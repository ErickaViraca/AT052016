	R.- Ruby have to ways to create comments one is inline 
	and the other is miltiline.

	- To create inline comments we just have to add this symbol 
	"#" at the beginning of a line. 
	- To create multiline comments we have to write between these 
	instruccions: =begin and =end

b) When you should use each type
	R.- When we can write a short line we use inline comments and 
	we want to write a paragraph we have to use multiline comments.
